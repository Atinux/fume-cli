# Usage
<!-- usage -->
```sh-session
$ npm install -g fume-cli
$ fume COMMAND
running command...
$ fume (-v|--version|version)
fume-cli/0.0.1 darwin-x64 node-v12.16.3
$ fume --help [COMMAND]
USAGE
  $ fume COMMAND
...
```
